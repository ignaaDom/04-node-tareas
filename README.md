#Notas:

Informacion sobre lo que puede hacer la aplicación: 
```
* Despliega un menú con diferentes opciones.
* Se pueden elegir entre las suguientes opciones: 
    - Crear Tarea
    - Listar tarea
    - Listar tarea completadas
    - Listar tarea pendientes
    - Completar tarea(s)
    - Borrar tarea
```
